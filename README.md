# Baby FlashCard

시스템 언어를 자동 감지해 **영어 + 시스템 언어**를 함께 보여주는 이미지 카드 앱입니다.

## 기능
- 브라우저 언어(`navigator.language`)를 기준으로 다국어 카드 표시
- 영어 단어 클릭 시 영어 음성 재생(Web Speech API)
- 시스템 언어 단어 클릭 시 해당 언어 음성 재생
- 단어별 음성 재생 버튼(영어 → 시스템 언어 순차 재생) 제공
- 카테고리별 효과음(카테고리 진입 시 브라우저 내장 오실레이터) 자동 재생
- 이전/다음 카드 전환 + 키보드 좌/우 이동
- 카드 전환 애니메이션 및 이미지 펄스 애니메이션
- 교통수단/날씨/집안 물건/놀이터 카테고리 구성
- 다수 단어에 직접 제작한 SVG 일러스트 추가(과일/동물/집안물건/놀이터/교통수단/날씨)

## 카테고리
- 과일
- 동물
- 해양동물
- 곤충
- 야채
- 직업
- 교통수단
- 날씨
- 집안 물건
- 놀이터

## 라이선스/소스 정책
- 기본 일러스트는 본 프로젝트에서 직접 제작한 SVG를 사용하고, 효과음은 외부 파일 없이 브라우저(Web Audio API)에서 생성합니다.
- 상업적 사용을 고려해 외부 무료 리소스 사용 시 아래 기준을 권장합니다.
  - `Free for commercial use`
  - `No attribution required` (또는 요구 시 출처 명시)
- 추천 사이트 예시: Unsplash, Pixabay, Pexels, Freepik, 공유마당, Pixabay Music, YouTube Audio Library 등

## 실행 방법
정적 웹 앱이므로 파일을 그대로 열거나 로컬 서버로 실행할 수 있습니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 접속.
